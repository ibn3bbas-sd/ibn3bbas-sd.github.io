# 📇 Ibn Abbas Mohammed Elhadi — Digital Contact Card

A professional, animated digital contact card built with pure HTML, CSS, and JavaScript. Hosted via GitHub Pages and accessible through a QR code embedded in my email signature.

🔗 **Live:** [ibn3bbas-sd.github.io/contact-card](https://ibn3bbas-sd.github.io/contact-card)

---

## ✨ Features

- **Glassmorphism design** — dark navy/teal theme with frosted glass card effect
- **Animated entrance** — staggered animations on page load for a premium feel
- **Save to phone** — 💾 Save buttons next to each number download a `.vcf` contact file instantly
- **Company link** — clicking the company name redirects to the official website
- **Email & LinkedIn links** — all contact details are tappable and clickable
- **QR code signature** — branded QR with company logo used as an email signature
- **Mobile responsive** — optimised layout for all screen sizes
- **Save as PDF** — print-friendly version with a single click

---

## 🗂️ Files

| File | Description |
|------|-------------|
| `index.html` | The contact card — all HTML, CSS, and JS in one file |
| `ibnabbas-signature-qr.png` | Branded QR code used in the email signature |

---

## 🚀 How It Works

1. QR code is embedded in the email signature
2. Recipient scans the QR → lands on this hosted contact card
3. They tap **Save** to download the `.vcf` and add the contact to their phone instantly
